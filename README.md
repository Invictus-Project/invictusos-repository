# InvictusOS APT Repository

Official package repository for InvictusOS.

## Add this repository
```bash
echo "deb https://invictus-project.github.io/invictusos-repository sagaris main" | sudo tee /etc/apt/sources.list.d/invictusos.list
sudo apt update
```
